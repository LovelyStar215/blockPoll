BLOCKPOLL

Immutable Elections on the Blockchain

blockPoll Mission Statement

blockPoll is a pseudo-anonymous, online, blockchain-based voting tool, used to facilitate organizational-based proxy voting and polling mechanisms for a wide array of uses.

Technology Key Points

Decentralised

Each certificate is stored as a smart contract. This adds a fundamental level of security, differentiating our storage method from any currently in existence.

Learn more

Smart Contracts

Each certificate is stored as a smart contract. This adds a fundamental level of security, differentiating our storage method from any currently in existence.

Learn more

Immutable

The immutable nature of blockchains means that information stored on them can never be changed. As a result, all certificates produced through our platform will exist, unchanged, forever.

Learn more

Public

We believe that public information should be exactly that; public. Everyone has equal access to certificate information.

Learn more

Issuer Managed

Asymmetric cryptography means that only the issuing institution can modify certificates created under their name.

Learn more

Verifiable

Certificates can be verified by validating who the signatory is. This means that no one can ever pretend to be another issuer as each signature is unique.

Learn more

User Friendly

Designed from the ground up to be as easy to use for both issuers and recipients alike. You don’t even need to interact with our system to have your information stored on it.

Learn more

Opensource

We embrace the open source mentality of the blockchain. All smart contracts and source code will remain open and publically verifiable.
